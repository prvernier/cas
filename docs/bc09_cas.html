cas_id

The attribute cas_id attribute is used in 5 of the 6 CASFRI attribute tables and is only described here. It has been modified slightly from the CAS_04 and is a concatenation of several attributes and strings (“bc08”, src_filename, map_id, feature_id, ogc_fid).

identification_id

  • The attribute identification_id is created from the attribute inventory_standard_cd:
    • mapvalues(inventory_standard_cd, c(“F”,“V”,“I”,“L”), c(4,5,6,7))
    • F = Forest Inventory Planning (FIP)
    • V = full VRI
    • I = incomplete VRI
    • L = Landscape Vegetation Inventory (LVI); low cost alternative to VRI
friList = c("F","V","I","L")
casList = c(4,5,6,7)
x = mutate(x, identification_id = mapvalues(inventory_standard_cd, friList, casList))
